# Project Title

ASCII Art Converter

## Description

ASCII Art Converter is a website that lets you convert images to ASCII art. Search for images in the giphy search tab. Click on any image that you want to convert to ASCII art.

## Built With

* Materialize 
* Giphy
* Filestack
